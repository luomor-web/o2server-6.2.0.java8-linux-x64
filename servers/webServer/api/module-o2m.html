    Module

    o2m

    o2m 是O2OA移动端APP提供的调用原生控件的能力，帮助开发者高效使用拍照、定位等手机系统的能力，同时可以直接使用扫一扫、打开原生应用、选择时间，人员，组织等业务的能力，带给门户接近原生代码的体验
    o2m 只有在O2OA移动端APP中才能提供能力

    Usable Range

    流程表单 门户

    Syntax

    // 可以在移动端 流程表单、门户 上使用
this.o2m
