    getTask() → {Task|Null}

    Summary

    获取当前流程与当前用户相关的待办对象：task对象。

    Description

    当前流程实例正在流转中，并且当前用户有待办，则返回当前用户的待办对象，否则返回null。

    var task = this.workContext.getTask();
